You might not need jQuery plugins

When youmightnotneedjquery you also might not need jQuery plugins. This list will start out with personal preferences, but since the goal is to have a generic list all input is welcome.

Credits:

The idea for this came from Lea Verou and was discussed at the lazyweb-request repo The groundwork for this was done by Zack Bloom & Adam Schwartz in the youmightnotneedjquery project
